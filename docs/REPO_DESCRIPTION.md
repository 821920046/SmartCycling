## 三、仓库长描述(可用于 README 顶部或介绍页)

> **智能骑行（SmartCycling）** 是一款面向安卓手机的智能骑行应用。它在开屏时自动搜索并连接迈金（Magene）S314 无线蓝牙速度/踏频传感器,随后进入高德地图界面;用户输入目的地并开始骑行后,App 自动规划骑行路线并切换到横屏骑行仪表盘——左侧实时导航,右侧大字号展示速度、骑行时长、骑行路程、平均速度与踏频。
>
> 项目采用 Kotlin + Jetpack Compose + MVVM 架构,蓝牙基于 Nordic Android-BLE-Library 接入标准 CSC（速度/踏频）服务,地图与导航基于高德开放平台,骑行记录与轨迹使用 Room 本地存储,并通过前台服务在锁屏下持续采集数据。
